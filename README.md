# TradeSphere

TradeSphere is a full-stack paper-trading platform for Indian markets focused on skill-building, discipline, and behavioral improvement.

## Live Preview

Preview URL: https://tradesphere-seven.vercel.app/

## Core Features

### Paper Trading Engine

- Simulated trading only (no real-money execution)
- Market, limit, and stop-loss order flows
- Intraday and overnight position modes
- Stock-level trading screens with charting and execution context

### Portfolio and Performance

- Real-time style virtual portfolio tracking
- Holdings, invested value, P&L, and win-rate metrics
- Portfolio history and trend visualization
- Best/worst performer insights and market snapshot panels

### Discipline Challenges

- 7-day beginner challenge and 30-day pro challenge
- Discipline-first leaderboard (score-based, not profit-based)
- Monthly champion system with countdown and recognition
- Challenge status integrated into dashboard and profile surfaces

### Competitions and Contest Rooms

- Live and upcoming competitions with join flow
- Competition detail pages with contest-specific leaderboard
- Notify-me flow and user contest suggestions
- Seamless transition from competition to paper trading

### Behavioral and AI Coaching

- Mood tracking and emotion-linked trade analysis
- Mistake tracking and pattern visibility
- AI coaching insights and improvement guidance

### Accountability and Community

- User-selected accountability partner requests
- Request/accept/decline partner workflow
- Weekly accountability reviews and summaries

### Pro Experience

- Pro-gated advanced experiences and challenge access
- Weekly report support and export capabilities
- Plan-aware UI gates across premium features

## Platform Modules

- Authentication and onboarding
- Trade terminal
- Portfolio and analytics
- Missions and learning progression
- Competitions and leaderboard system
- Discipline challenges and champion cycle
- Accountability partner network

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- NextAuth
- Recharts and lightweight-charts
